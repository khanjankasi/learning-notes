## Security Playbook

A standardized incident response process.

Purpose:

- Guide response actions
- Improve consistency
- Accelerate threat handling

Example:

1. Detect attack
2. Investigate
3. Contain
4. Recover
5. Document

---

## Penetration Testing

Authorized simulated cyberattack performed to identify vulnerabilities.

Purpose:

- Find weaknesses before attackers do
- Improve security posture

Also known as:

- Ethical Hacking
