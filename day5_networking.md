## Topics Covered

### Module 5 - Routers, Switches and Access Points

#### Router

Connects different networks together.

Example:

Home Network
↓
Router
↓
Internet

Responsibilities:

* Route traffic
* Connect networks
* Perform Network Address Translation (NAT)

---

#### Switch

Connects devices within the same network.

Examples:

* Computers in a lab
* Devices in an office

---

#### Access Point

Provides wireless connectivity to devices.

Examples:

* Wi-Fi routers
* Enterprise wireless systems

---

### Module 6 - Packets and Protocols

Data is transmitted across networks using packets.

Process:

Large File
↓
Packets
↓
Transmission
↓
Reassembly

---

### Protocols

#### TCP (Transmission Control Protocol)

Characteristics:

* Reliable
* Ordered delivery
* Error checking

Used for:

* Web browsing
* File transfers

---

#### UDP (User Datagram Protocol)

Characteristics:

* Faster
* No delivery guarantees

Used for:

* Online gaming
* Video streaming
* Voice calls

---

### HTTP and HTTPS

#### HTTP

Protocol used to transfer web pages.

#### HTTPS

Secure version of HTTP using encryption.

Benefits:

* Privacy
* Data integrity
* Secure communication

---

### Module 7 - Network Security Fundamentals

#### Firewall

Monitors and filters network traffic.

Purpose:

* Block unauthorized access
* Protect networks

---

#### VPN (Virtual Private Network)

Creates an encrypted tunnel over the Internet.

Benefits:

* Privacy
* Secure remote access

---

#### IDS (Intrusion Detection System)

Detects suspicious activity.

---

#### IPS (Intrusion Prevention System)

Detects and blocks suspicious activity.

---

## Key Takeaways

* Routers connect networks.
* Switches connect devices within a network.
* DNS translates names into IP addresses.
* TCP prioritizes reliability.
* UDP prioritizes speed.
* HTTPS provides secure communication.
* Firewalls and VPNs improve network security.

## Reflection

Networking knowledge provides the foundation for cybersecurity, cloud computing, system administration, and software engineering.
