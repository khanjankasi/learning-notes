## Topics covered:

### Module 1 - What is a Network?

A network is a collection of devices connected together to share data and resources.

Examples:

* Home Wi-Fi network
* College computer lab
* The Internet

### Types of Networks

#### LAN (Local Area Network)

A network covering a small geographical area.

Examples:

* Computer lab
* Office network
* Home network

#### WAN (Wide Area Network)

A network covering a large geographical area.

Examples:

* Internet
* Corporate networks connecting multiple branches

---

### Module 2 - How the Internet Works

The Internet is a global network of interconnected networks.

Typical communication flow:

User Device
↓
Router
↓
Internet Service Provider (ISP)
↓
Destination Server
↓
Response Returned

Key Concepts:

* Client
* Server
* Request
* Response

---

### Module 3 - IP Addresses

An IP address uniquely identifies a device on a network.

Example:

192.168.1.10

#### Types of IP Addresses

##### Private IP

Used inside local networks.

Examples:

* 192.168.x.x
* 10.x.x.x

##### Public IP

Visible on the Internet.

Assigned by an ISP.

##### Static IP

Remains unchanged.

##### Dynamic IP

Assigned automatically and may change over time.

---

### Module 4 - DNS (Domain Name System)

DNS translates domain names into IP addresses.

Example:

google.com
↓
IP Address

Purpose:

* Easier navigation
* Human-friendly naming system

---

## Key Takeaways

* Networks connect devices for communication.
* LANs cover small areas while WANs cover large areas.
* Every device requires an IP address.
* DNS converts domain names into IP addresses.
* The Internet uses a client-server model.

## Reflection

Networking forms the foundation of cybersecurity, cloud computing, and modern software systems.
