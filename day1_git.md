# Day 1 - Git & GitHub

## What I Learned

* Git is a version control system.
* GitHub hosts repositories online.
* A repository stores project files and their history.
* `git status` shows the current state of the repository.
* `git add` stages changes.
* `git commit` creates a snapshot of changes.
* `git log --oneline` displays commit history.
* `origin` is the default name for a remote repository.
* `git push` uploads local commits to GitHub.

## Commands Practiced

```bash
git status
git add notes.txt
git commit -m "Add first notes"
git log --oneline
```

## Key Takeaway

Git helps track changes, manage versions, and collaborate efficiently on projects.
